Edge Testing Tool

The Edge Testing Tool is a collection of testing utilities created to validate the requirements of the ONC 2014 and 2015 Edition Health IT Certification Program. The Edge Testing Tool was originally designed to test only network "Edge" capabilities, but over time assumed HISP and other transport testing abilities, along with C-CDA and content validation utilities. The Edge Testing Tool software is open source and available for download.
